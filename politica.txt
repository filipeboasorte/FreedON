1. INTRODUÇÃO

O FreedON está comprometido com a proteção de dados pessoais e a privacidade de seus usuários, atuando em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 – LGPD) e demais normas aplicáveis.

Esta Política estabelece, de forma transparente, como ocorre o tratamento de dados pessoais no contexto do aplicativo.

2. ESCOPO

Esta Política se aplica a todos os usuários que utilizam o aplicativo FreedON, incluindo:

Usuários cadastrados
Usuários visitantes (quando aplicável)
Dispositivos Android
3. CATEGORIAS DE DADOS COLETADOS
3.1 Dados cadastrais
Nome completo
E-mail
Telefone
Senha (armazenada de forma criptografada)
Informações de perfil
3.2 Dados de autenticação
Credenciais de login
Tokens de acesso
Histórico de sessões
3.3 Dados de geolocalização
Localização precisa ou aproximada
Coletada exclusivamente durante o uso ativo do aplicativo (primeiro plano)
3.4 Dados de mídia (Câmera e Galeria)
Fotos capturadas pelo usuário
Arquivos enviados
3.5 Dados técnicos e de uso
Endereço IP
Identificadores do dispositivo
Sistema operacional
Logs de acesso
Tempo de uso e interações
4. FINALIDADES DO TRATAMENTO

Os dados são tratados para:

Criação e gestão da conta do usuário
Autenticação e segurança de acesso
Execução dos serviços oferecidos
Personalização da experiência
Comunicação com o usuário
Prevenção a fraudes e abusos
Cumprimento de obrigações legais
5. BASES LEGAIS (ART. 7º LGPD)

O tratamento de dados ocorre com base em:

Execução de contrato (Art. 7º, V) – criação e uso da conta
Consentimento (Art. 7º, I) – localização, câmera e galeria
Legítimo interesse (Art. 7º, IX) – segurança, melhoria do app
Obrigação legal (Art. 7º, II)
6. COMPARTILHAMENTO DE DADOS

O FreedON poderá compartilhar dados com:

6.1 Operadores de dados
Serviços de cloud (ex: AWS, Google Cloud, Firebase)
Ferramentas de analytics
Serviços de autenticação
6.2 Autoridades públicas

Quando exigido por lei ou ordem judicial

 Não comercializamos dados pessoais.

7. TRANSFERÊNCIA INTERNACIONAL

Os dados podem ser transferidos para fora do Brasil, garantindo:

Cláusulas contratuais adequadas
Adoção de padrões internacionais de segurança
Conformidade com a LGPD
8. RETENÇÃO DE DADOS

Os dados serão armazenados:

Enquanto a conta estiver ativa
Pelo tempo necessário para cumprir obrigações legais
Para exercício regular de direitos

Após isso, serão:

Excluídos
Anonimizados
9. SEGURANÇA DA INFORMAÇÃO

Adotamos medidas robustas, incluindo:

Criptografia (em trânsito e em repouso)
Controle de acesso baseado em função (RBAC)
Autenticação segura
Monitoramento contínuo
Logs e auditoria
10. DIREITOS DOS TITULARES (ART. 18 LGPD)

O usuário poderá solicitar:

Confirmação de tratamento
Acesso aos dados
Correção
Anonimização ou exclusão
Portabilidade
Revogação do consentimento
Informação sobre compartilhamento
